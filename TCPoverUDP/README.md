# TCP over UDP
Pseudo-TCP socket over UDP.<br/>
## Functionality
<br/>"main" fucntion job:
1. Creates threads for server, client.
2. Makes connection between client and server.
3. Send 100 string from client to server(server can drop packets).
4. Closes connection.
